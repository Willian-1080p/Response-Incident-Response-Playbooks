# Incident Response Playbooks

Playbooks praticos para resposta a incidentes de seguranca, com foco em triagem, contencao, erradicacao, recuperacao e licoes aprendidas.

## Playbooks iniciais

- `playbooks/phishing.md`
- `playbooks/ransomware.md`
- `playbooks/brute-force.md`

## Como usar

1. Identifique o tipo de incidente.
2. Abra o playbook correspondente.
3. Registre evidencias usando `templates/incident-report.md`.
4. Execute apenas acoes autorizadas no ambiente afetado.
5. Atualize a secao de licoes aprendidas apos o encerramento.
