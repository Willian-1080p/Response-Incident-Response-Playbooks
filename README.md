# Incident Response Playbooks

Coleção de playbooks voltados para Resposta a Incidentes de Segurança da Informação.

## Objetivo

Este repositório foi criado para documentar procedimentos padronizados de resposta a incidentes, permitindo identificação, contenção, erradicação, recuperação e lições aprendidas para diferentes cenários de segurança.

## Estrutura

```text
Incident-Response-Playbooks/
├─ playbooks/
│  ├─ phishing-response.md
│  ├─ malware-response.md
│  ├─ ransomware-response.md
│  ├─ data-breach-response.md
│  └─ insider-threat-response.md
├─ templates/
│  ├─ incident-report.md
│  ├─ timeline.csv
│  └─ evidence-log.md
└─ data/
   └─ incident-types.json
```

## Playbooks Disponíveis

### Phishing Response

Procedimentos para análise e contenção de campanhas de phishing.

### Malware Response

Procedimentos para tratamento de infecções por malware.

### Ransomware Response

Fluxo de resposta para ataques de ransomware.

### Data Breach Response

Tratamento de vazamentos e exposição de dados.

### Insider Threat Response

Investigação de incidentes causados por usuários internos.

## Fluxo Geral de Resposta

1. Identificação
2. Classificação
3. Contenção
4. Erradicação
5. Recuperação
6. Lições Aprendidas
